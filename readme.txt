== Description ==

Free customizable popup with basic functionality. Supports Facebook, Instagram, Twitter, YouTube, LinkedIn and Snapchat. You can also put your own code. The plugin is SEO friendly.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/easy-free-popup` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Easy Free Popup screen to configure the plugin
4. For testing purposes it can be helpful to remove cookie "efp_modal_cookie" after requests
